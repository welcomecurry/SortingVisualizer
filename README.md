## Sorting Visualizer

React app that visualizes sorting algorithms.

| Sorting Algorithms  | Time Complexity |
| ------------- | ------------- |
| Merge Sort    | O(n log(n))   |
| Quick Sort    | O(n log(n))   |
| Bubble Sort   | O(n^2)        |
| Insertion Sort| O(n^2)        |
| Selection Sort| O(n^2)        |
